Este proyecto no se encuentra en produccion. Siemple app que intenta
establecer una conexion con un motor de base de datos, generar Query's.
Operaciones CRUD en GUI, implementando Java, jetty server embebido,
javascript(jquery), html y css(utilizando materialize css)

Simula la conexion remota de una placa arduino uno, para guardar mediante sensores
conectados a la misma, alguna unidad de medida.Guardando datos en la -
app(request and response).

Si necesita el dump de la base de datos, por favor envíeme su correo, se
lo facilitaré.

La documentación(la api de las clases de la app) se encuentra en doc/index.html-
Revísela con atención, habrá muchas especificaciones tecnicas allí.
